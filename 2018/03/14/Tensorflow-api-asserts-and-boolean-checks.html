tf.assert_proper_iterable

功能

判断 values 是不是一个合适的迭代器，如果 values 是一个 Tensor, SparseTensor, np.array, tf.compat.bytes_or_texts_types 则报错。或者不是迭代器也报错。
这个函数时有用处的，因为张量 Tensor 或者数组 np.array 本身也是可迭代的 (iterable).
